## Setup
To run this project, just navigate to the project folder and open the HTML file.
```
# Clone this repository
$ git clone https://github.com/bpa03/quotes-generator.git

# Go into the repository
$ cd ./quotes-generator

# Install dependencies
$ npm install

# Run the app
$ npm start
```
